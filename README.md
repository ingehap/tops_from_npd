# tops_from_npd

## Author:
Trond Anders Seland

## Main data file is from NOD:
https://factpages.sodir.no/en/strat/TableView/Wellbores

## Purpose:
The script reads an NPD (Norwegian Petroleum Directorate / Sokkeldirektoratet) `strat_litho_wellbore.csv` plus a well-list file, 
then writes one tops CSV per stratigraphic level (GROUP, FORMATION, MEMBER).
